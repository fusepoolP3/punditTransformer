Preview

  1. - service in module Pundit2.Preview

Handles the preview of an item: given an item, show/hide a preview of some details of the item. It is possible set an item as sticky or not, and check if a given item is set as sticky or not.

Methods

  • getWelcomeHeaderMessage();

    Return the welcome message that will be shown in the header of preview panel when no preview is shown.

    Returns

    string

    header welcome message set in the configuration of Preview module. See here for details

  • getWelcomeBodyMessage();

    Return the welcome message that will be shown in the body of preview panel when no preview is shown.

    Returns

    string

    body welcome message set in the configuration of Preview module. See here for details

  • showDashboardPreview(Item);

    Show a preview of the given item in the chosen panel. The panel where preview will be show is that set in configuration object. See here for details Preview will show the follow details of an item:

    • label
    • types
    • description (if present)
    • image (if present)

    Parameters

    Param Type Details
    Item Object

    to show in preview panel.

  • getItemDashboardPreview();

    Get current item shown in preview preview

    Returns

    Object

    Current item shown in preview panel

  • setItemDashboardSticky(Item);

    Set an item as sticky.

    In this way, if there is not a current item shown in preview, the panel in dashboard will not be empty but the sticky item will be shown on it.

    Parameters

    Param Type Details
    Item Object

    to set as sticky.

  • getItemDashboardSticky();

    Get the current stitky item

    Returns

    Object

    Current item set as sticky

  • clearItemDashboardSticky();

    Reset dashboard preview, cleaning both item show in preview and sticky item.

    In this way, preview panel will be empty and will be shown welcome message

  • hideDashboardPreview();

    Hide current item preview. If there is a sticky item, it will be shown in preview panel, otherwise preview panel will be empty and will be shown welcome message

  • isStickyItem(Item);

    Check if the passed item it's the current sticky item or not.

    In this way, if there is not a current item shown in preview, the panel in dashboard will not be empty but the sticky item will be shown on it.

    Parameters

    Param Type Details
    Item Object

    item to check if it is the sticky item.

    Returns

    boolean

    true if the passed item is the current sticky item, false otherwise
