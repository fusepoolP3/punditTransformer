KORBODEFAULTCONF

  1. - object in module KorboEE

This is the object configuration in which is possible to set values of its properties.

Each properties has a default value. You can see an example of configuration object here Example It is possible to make a different configuration, depending on the usage you want. To see possible usage and its configuration, see Tafony Compatibility and Use Only Callback

Properties

  • endpoint

    string

    URL where Korbo server is installed

    Default value:

     endpoint: "http://demo-cloud.api.korbo.org/v1/"

  • basketID

    integer

    Korbo basket ID where a new entity is saved

    Default value:

     basketID: 1

  • updateTime

    integer

    Minimal wait time after last character typed before search starting

    Default value:

     updateTime: 1000

  • labelMinLength

    integer

    Minimal number of characters that needs to be entered before search starting

    Minimal length of label of new entity: to create a new entity, the label must have labelMinLength

    Default value:

     labelMinLength: 3

  • copyToKorboBeforeUse

    boolean

    Set to true if want to copy an entity in Korbo and use it. False otherwise.

    Default value:

     copyToKorboBeforeUse: false

  • tafonyId

    string

    Id of the HTML input text Must be set only to use widget in Tafony Compatibility For more details see Tafony Compatibility

    Default value:

     tafonyId: 'default_tafony_id'

  • tafonyName

    string

    Name of the HTML input text

    Must be set only to use widget in Tafony Compatibility For more details see Tafony Compatibility

    Default value:

     tafonyName: 'default_tafony_name'

  • nameInputHiddenUri

    string

    Hidden input name containing entity location

    For more details see Tafony Compatibility

    Default value:

     nameInputHiddenUri: 'default_korbo_uri'

  • nameInputHiddenLabel

    string

    Hidden input name containing entity label

    For more details see Tafony Compatibility

    Default value:

     nameInputHiddenLabel: 'default_korbo_label'

  • useTafonyCompatibility

    boolean

    Set to true to use widget in Tafony Compatibility

    For more details see Tafony Compatibility

    Default value:

     useTafonyCompatibility: false

  • useOnlyCallback

    boolean

    Set to true to use widget only with callback

    See Use Only Callback for details

    Default value:

     useOnlyCallback: false

  • globalObjectName

    string

    Name for exposed global object. This object is used to call API callback and to register callbacks to get called when some events are fired.

    For details about API Callbacks and events see EE

    Default value:

     globalObjectName: 'EE'

  • onLoad

    function

    This function has to be defined when mode to use the widget is useOnlyCallback.

    The function will be get called when widget is ready to use.

    For details about useOnlyCallback usage see Use Only Callback

    Default value:

     onLoad: null

  • searchTypes

    array

    The array has to contain URI of types to use for searching an entity

    Default value:

     searchTypes: ['http://person.uri', 'http://philosopher.uri', 'http://place.uri']

  • limitSearchResult

    integer

    Number or result per page

    Default value:

     limitSearchResult: 10

  • providers

    object {provider_name: true/false}

    List of available provider where to search in. Object must contain the name of provider and a state for each one: true if you want to use that provider to searchin, false otherwise

    Default value:

    providers: {
    freebase: false,
    dbpedia: false
}

  • type

    array of object

    Types available to select when you want create a new entity

    Each object should represent a type and has to contain

    • label: label shown in widget
    • state: true if you want to show type selected yet, false otherwise
    • URI: URI of the type

    Default value:

     type: [
   {
    label: 'Schema.org place',
    state: true,
    URI:'http://schema.org/Place'
   },
   {
    label: 'FOAF person',
    state: false,
    URI:'http://xmlns.com/foaf/0.1/Person'
   },
   {
    label: 'W3 Spatial thing',
    state: true,
    URI:'http://www.w3.org/2003/01/geo/wgs84_pos#SpatialThing'
   }
 ]

  • languages

    array of object

    Language available to select when you want create a new entity

    Each object should represent a type and has to contain

    • name: language name, show in select input
    • value: value to send in the http request
    • state: true to set this language as default, false otherwise

    Default value:

     languages: [
   {
    name:'Italian',
    value:'it',
    state: false
   },
   {
    name:'English',
    value:'en',
    state: true
   },
   {
    name:'German',
    value:'de',
    state: false
   }
 ]

  • visualizeCopyButton

    array of string

    Language available to select when you want create a new entity List of providers from which copy and modify an entity, to create a new one in korbo.

    Each object should represent a type and has to contain

    • name: language name, show in select input
    • value: value to send in the http request
    • state: true to set this language as default, false otherwise

    Default value:

        visualizeCopyButton: ['freebase']

  • onReady

    function

    The function will be get called when widget is ready to use.

    Default value:

     onReady: null

  • autoCompleteMode

    string

    Define wich mode of autocomplete use. Each mode differs on how the results are shown.

    • simple mode visualize only label of found entities
    • full mode visualize the label, the depiction and the first type of found entities

    Default value:

     autoCompleteMode: 'simple'

  • autoCompleteOptions

    string

    Define which buttons to show in autocomplete visualization.

    • none no buttons are shown
    • search will be show a Search on LOD button, that will open widget in Search window to find entity in other LOD providers
    • new will be show a Create New button, that will open widget in New window, where is possible to create a new entity in Korbo
    • all display both buttons, the 'Create New' and the 'Search on LOD'

    Default value:

     autoCompleteOptions: 'none'
