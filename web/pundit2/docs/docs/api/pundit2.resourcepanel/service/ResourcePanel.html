ResourcePanel

  1. - service in module Pundit2.ResourcePanel

Handle the resource panel visualization and its content.

Given a triple (complete or not), il will be show the list of items from their provenience ('Page Items', 'My Items', 'Vocabularies')

A triple is an array of URI, where:

  • in the first position there is the Subject URI
  • in the second position there is the Predicate URI
  • in the third position there is the Object URI

A triple is defined as complete when all URI are defined, incomplete otherwise.

    Open a popover containing a textarea where is possible to insert a literal.

  • showItemsForSubject(triple, target, label);

    Open a popover where subject items are shown grouped according from their provenance ('Page items', 'My items', 'Vocabularies'). If predicate is not defined in the triple, all items will be shown. If predicate is defined in the triple, will be shown only items whose types are compatible with predicate domain

    My items are visible only if user is logged in.

    It is possible to insert a label to filter shown items and start a searching of new items in the vocabularies.

    Parameters

    Param Type Details
    triple Object

    object (for details content of this object, see here)
    target DOMElement

    DOM Element where to append the popover
    label string

    label used to search subject in the vocabularies and filter shown Page Items and My Items

    Returns

    Promise

    return a promise that will be resolved when a subject is selected
