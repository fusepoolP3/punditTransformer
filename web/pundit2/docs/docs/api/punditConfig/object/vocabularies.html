vocabularies

  1. - object in module punditConfig

Array of url Specifies relations vocaularies that will be available to Pundit users (defines a list of relations with domain and ranges). Each vocabulary definition is a JSONP file available on the Web and is loaded by resolving an absolute URL.

Default:

 vocabularies: []
