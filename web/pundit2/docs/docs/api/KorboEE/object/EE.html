EE

  1. - object in module KorboEE

Global Object exposed by the Entity Editor to interact with callbacks and features.

Methods

  • callOpenSearch([label]);

    Open the modal on 'Search' page and start searching the given string.

    If no string is supplied, open the modal on 'Search' page and no search get start.

    Parameters

    Param Type Details
    label
    (optional)
    		 String

    Reference to search.

  • callOpenNew([entity]);

    Open the modal on 'New' page.

    If an object is supplied, the fields of the form are filled with object values. If no object is supplied, the fields of the form are empty

    Parameters

    Param Type Details
    entity
    (optional)
    		 Object

    An entity to create. It has to be, optionally, the follow properties:

    • value: entity URL
    • label: entity label
    • type: array of types
    • image: entity depiction URL
    • description: abstract of entity
    • language: language value of entity to create

  • callEdit([entity]);

    Open the modal on 'New' page.

    The fields of the form are filled with all values of entity to edit.

    Parameters

    Param Type Details
    entity
    (optional)
    		 String

    ID to edit.

  • callCancel();

    If a modal is open, close the modal

  • callCopyAndUse(entity);

    Given a non-korbo entity, copy that entity in Korbo and return the new one.

    Parameters

    Param Type Details
    entity Object

    entity to copy in Korbo and use.

  • onCancel(callback);

    Register a callback function on the Cancel event. When the modal is closed, Cancel event is fired and registered function will get called

    Parameters

    Param Type Details
    callback function

    The callback function gets called when the Cancel event is fired

  • onOpen(callback);

    Register a function on the Open event. When the modal is opened, the Open event is fired and registered function will get called

    Parameters

    Param Type Details
    callback function

    The callback function gets called when the Open event is fired

  • onSave(callback);

    Register a callback function on the Save event. A Save event is fired when a new entity is saved, or copied or just used. To the registered function will be passed the use/saved entity with the entity parameter.

    Parameters

    Param Type Details
    callback function

    The callback function gets called when the Save event is fired, passing the entity object which is the saved/used semantic entity having the following properties:

    • value: entity URL
    • label: entity label
    • type: array of types
    • image: entity depiction URL
    • description: entity abstract
    • language: entity language
